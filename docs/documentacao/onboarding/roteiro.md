# Roteiro de Onboarding

Guia de integração para novos contribuidores do GovHub BR. Antes de iniciar
uma contribuição, siga a etapa de [Instalação](../instalacao.md) para preparar
e explorar o ambiente local.

## Primeiro Dia

1. **Ler** o [README do projeto](https://github.com/GovHub-br/data-application-gov-hub)
2. **Concluir** a [Instalação](../instalacao.md) para preparar o ambiente
3. **Conhecer** os repositórios e a arquitetura geral do projeto
4. **Configurar** a assinatura de commits e o fluxo Git
5. **Identificar** com o time a primeira atividade de contribuição

## Primeira Semana

1. **Entender** a [Arquitetura Medallion](../arquitetura/medallion.md)
2. **Consultar** os tutoriais técnicos do Onboarding conforme sua área
3. **Configurar** Git com GPG ([Git Workflow](git-workflow.md))
4. **Ler** os padrões de engenharia e contribuição do projeto
5. **Abrir** seu primeiro PR com uma melhoria pequena
